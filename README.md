# AI修图 App

聊天式智能修图工具，发图片 + 说需求，AI自动帮你修图。

## 架构

```
┌──────────────┐     HTTP/Multipart     ┌──────────────┐     CLI      ┌─────────┐
│  Flutter App  │ ───────────────────→  │ Python Flask  │ ──────────→ │ Meitu   │
│  (聊天界面)   │ ←───────────────────  │ 后端服务      │ ←────────── │ API     │
└──────────────┘     结果图片+积分       └──────────────┘              └─────────┘
```

## 功能

- 📱 **聊天式体验**：像聊天一样发图片+文字需求
- 🤖 **AI理解需求**：自动识别你要抠图、换背景、去水印、改字还是滤镜
- ✂️ **美图API驱动**：专业修图能力驱动
- 💳 **积分显示**：每次处理显示消耗和剩余积分

## 支持的操作

| 操作 | 示例说法 |
|------|---------|
| 抠图 | "把这个人抠出来"、"去背景" |
| 换背景 | "把背景换成海边"、"换背景" |
| 去水印 | "去掉上面的水印" |
| 改字 | "把'你好'改成'再见'" |
| 通用编辑 | "给我加个复古滤镜"、"调亮一点" |

## 本地开发

### 后端
```bash
cd server
pip install -r requirements.txt
python app.py
```

### Flutter App
```bash
flutter pub get
flutter run
```

## 后端配置

后端默认端口 `5078`，可通过 `PORT` 环境变量修改。

后端需要已配置好美图CLI凭证（`~/.meitu/credentials.json`）。
